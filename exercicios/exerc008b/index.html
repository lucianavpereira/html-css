Código-fonte / Pré-formatação

O comando document.getElementById('teste') é escrito em linguagem JavaScript.

O comando document.getElementById('teste') é escrito em linguagem JavaScript.

Observe que antes de envelopar o comando dentro da 'tag code' o espaçamento das letras é desigual. Chamamos de fonte monoespaçada, quando usamos a tag code para igualar os espaçamentos das letras.

num = int(input('Digite o número:')) if num % 2 == 0: print(f'O número {num} é PAR') else: print(f'O número {num} é IMPAR) print ('Fim do Programa')

Note que acima a visualização do código está uma ao lado da outra, isso porque a tag code por si só, não faz a identação correta.

        
        num = int(input('Digite o número:'))
        if num % 2 == 0:
            print(f'O número {num} é PAR')
        else:
            print(f'O número {num} é IMPAR)
        print ('Fim do Programa')

Nesse exemplo em que o código 'é impresso', visualmente notamos a identação do código. isso, porque a tag code foi envelopada dentro da tag pre.
