Principais Formatações de textos

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântico).

Nesta frase temos um termo em destaque usando a tag STRONG (semântico).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag I (não semântico).

Nesta frase temos um termo em ênfase usando a tag EM (semântico).

Texto Marcado

Podemos também criar um texto marcado usando a tag MARK.

Por padrão a tag mark, marca o texto de amarelo. Se quisermos mudar a cor usamos STYLE, comando de CSS para isso.

Texto Grande e Texto Pequeno

Estamos criando um Texto Grande e um Texto Pequeno, nesse parágrafo.

Obs. Repare que a tag BIG ficou destacada em vermelho. Isso, porque já é uma tag considerada obsoleta. Hoje se quiser criar um texto ou um trecho de texto com letras grandes, usa-se as folhas de estilo das CSS.

Texto Deletado

Podemos marcar um texto como deletado ou excluído para indicar que ele deve ser lido, mas não considerado.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o texto Sublinhado com a tag U (não semântica).

Texto Sobrescrito

Para inserir coisas do tipo X20 + 3.

Texto Subscrito

Para inserir coisas do tipo H2O.
